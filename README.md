# 🧩 Project Name: Tiles Gallery

## 📌 Project Overview
**Tiles Gallery** is a modern web application built with Next.js that allows users to explore, view, and manage a visually appealing collection of tiles/images. It focuses on clean UI, smooth user experience, and dynamic interactions.

---

## 🎯 Purpose
The purpose of this project is to:
- Practice full-stack development using Next.js
- Implement authentication and database integration
- Build a responsive and interactive gallery UI
- Manage user profiles and dynamic content

---

## 🌐 Live URL
🔗 https://my-tiles-gallery-webapp.vercel.app

---

## ✨ Key Features
- 🔐 User Authentication (Login / Logout)
- 👤 User Profile with update functionality (name & image)
- 🖼️ Dynamic Tiles/Image Gallery
- ⚡ Fast and optimized performance with Next.js
- 🎨 Modern UI with Tailwind CSS, HeroUI, & DaisyUI
- 🔔 Toast notifications for user feedback
- 📦 Form handling with React Hook Form
- 🔄 Loading spinners and smooth animations

---

## 🛠️ Technologies Used
- **Framework:** Next.js 16  
- **Frontend:** React 19  
- **Styling:** Tailwind CSS, DaisyUI, HeroUI  
- **Authentication:** Better Auth  
- **Database:** MongoDB  
- **Form Management:** React Hook Form  
- **Icons:** React Icons, Gravity UI Icons  
- **Animations:** Animate.css, React Fast Marquee  
- **Notifications:** React Toastify
